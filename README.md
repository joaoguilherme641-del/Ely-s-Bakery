# Ely-s-Bakery
Repositório da Ely's Bakery 
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Descrição: A Ely's Bakery é uma padaria de pequeno porte que busca organizar melhor seus processos internos, como controle de estoque, registro de pedidos, gestão de funcionários e acompanhamento do fluxo de caixa. O sistema foi desenvolvido com o objetivo de ajudar a avó do nosso cliente, oferecendo uma forma simples, rápida e prática de gerenciar as atividades da padaria. A proposta é facilitar o dia a dia no estabelecimento, reduzindo erros manuais e centralizando todas as informações em um único sistema eficiente e fácil de usar.
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Principais tecnologias: canva, draw.io, word, netbeans e excel
